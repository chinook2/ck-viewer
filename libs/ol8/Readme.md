# ck-viewer/libs/ol8

## OpenLayers 8.1.0

Files : 
- ol.css : style
- ol.js : production dist
- ol-patch.js : few add-on / overrides for specific behaviours


Use package.json to include sources :

    "js": [
        {
            "path": "libs/ol8/ol-debug.js"
        }
    ],
    "css": [
        {
            "path": "libs/ol8/ol.css"
        }
    ]
